## Installation

Setup article for linux: https://shom.dev/start/using-kanata-to-remap-any-keyboard
```sh
cargo install kanata
```

```sh
# On mac we need to run it with sudo
sudo kanata -c ~/.config/kanata/kanata.kbd
```
